Start
 ↓
Input: entropy list `fl`, number of runs `runs`, number of (x,y) pairs per run `n`
 ↓
Set θ_list = [0°, 5°, 10°, ..., 180°]
 ↓
For each θ in θ_list:
 ├─► Compute expected match %: exp = cos²(θ / 2)
 │
 ├─► Initialize total match accumulator: total = 0
 │
 ├─► Repeat for `runs` times:
 │   ├─► Choose random start index `start` in `fl` such that:
 │   │     start ≤ len(fl) - 2 * n
 │   │
 │   ├─► For j from 0 to n - 1:
 │   │   ├─► x = fl[start + 2*j]
 │   │   ├─► y = fl[start + 2*j + 1]
 │   │   ├─► if x < exp:
 │   │   │     a = b = 0 if y < 0.5 else 1
 │   │   │   else:
 │   │   │     a, b = (0, 1) if y < 0.5 else (1, 0)
 │   │   ├─► if a == b: match_count += 1
 │   │
 │   ├─► Compute run match % = (match_count / n) * 100
 │   ├─► Add to total: total += match %
 │
 ├─► Compute average match % for this θ: avg = total / runs
 ├─► Store:
 │     ├─► angle → θ
 │     ├─► expected value → cos²(θ / 2)
 │     └─► measured avg → avg
 ↓
After all angles:
 ├─► Plot:
 │     ├─► X-axis: θ (in degrees)
 │     ├─► Y-axis: match %
 │     ├─► Line 1: Theoretical curve (cos²)
 │     └─► Line 2: Measured from entropy
 ↓
FINISHED.
'''
