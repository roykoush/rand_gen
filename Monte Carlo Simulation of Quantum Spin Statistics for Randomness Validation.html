  Initialize total match accumulator: total = 0
 
  Repeat for `runs` times:
     Choose random start index `start` in `fl` such that:
         start  len(fl) - 2 * n

        x = fl[start + 2*j]

            a = b = 0 if y < 0.5 else 1
          else:
            a, b = (0, 1) if y < 0.5 else (1, 0)

     Add to total: total += match %
 
  Compute average match % for this : avg = total / runs
  Store:
       angle  
       expected value  cos( / 2)
       measured avg  avg
 
After all angles:
  Plot:
       X-axis:  (in degrees)
       Y-axis: match %
       Line 1: Theoretical curve (cos)
       Line 2: Measured from entropy
 
FINISHED.
